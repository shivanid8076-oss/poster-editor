# PosterForge Studio - Real Estate Ad Editor

A modern, web-based image editor specifically designed for creating stunning real estate posters, property flyers, and personal ads. Built with Fabric.js and pure CSS/JS for a lightweight, fast, and premium experience.

## ✨ Features

- **🎯 Real Estate Focus**: Custom presets for common real estate needs (Rishi Dixit contact, Property Deals).
- **✂️ Robust Image Cropping**: Interactive cropping tool that handles resizing and centering.
- **🎨 Canvas Color & Backgrounds**: Easy management of canvas background colors and "Cover-style" filling.
- **📱 Phone & Desktop Friendly**: Fully responsive design with special mobile touch-optimized sheets.
- **🎭 Layers & Styles**: Easy-to-use layer management, typography, shapes, and filters.
- **⚡ Fast & Light**: No heavy frameworks, just vanilla JS and CSS.

## 🚀 Getting Started

1.  Clone the repository: `git clone https://github.com/shivanid8076-oss/poster-editor.git`
2.  Open `index.html` in any modern web browser.
3.  Start creating your posters!
